Setup instructions
  1. Open Claude Desktop and go to Settings → Developer → Edit Config
  2. Add the block below to your claude_desktop_config.json
  3. Restart Claude Desktop
claude_desktop_config.json 
"mcpServers": {
  "emailzap": {
    "command": "npx",
    "args": ["-y", "@emailzap/mcp"],
    "env": {
      "EMAILZAP_TOKEN": "ezm_a3f9k2x8p1q7n5r4t6y0w2e8m3j5v1c"
    }
  }
}
